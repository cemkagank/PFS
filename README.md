# Fluid Simulations

## Build Instructions

### Fedora 
```
sudo dnf install raylib raylib-devel
git clone https://github.com/cemkagank/PFS
cd pfs
make make
```

### Ubuntu & Derivatives
Check https://github.com/raysan5/raylib/wiki/Working-on-GNU-Linux to get raylib, clone the repo same as fedora.
